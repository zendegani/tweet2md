XClipper 是一款适用于 X（Twitter）的免费、源码可用网页剪藏工具——一款 Chrome 扩展，可将线程、帖子和文章下载为 PDF 或干净的 Markdown，用于 Obsidian、研究、AI 工作流和离线归档。完全在浏览器中运行：无需 API 密钥、无需账号、无需跟踪。

一键导出内容：

- 保存为 Markdown
- 导出为 PDF（可选择文本、可点击链接、无栅格化）
- 复制 Markdown 到剪贴板
- 直接发送到 Obsidian
- 在 .md 文件旁本地下载图片

所有内容都在你的设备本地运行。无后端服务、无遥测、除 x.com 本身外无任何外部请求。

主要功能：

- 导出推文、线程、引用推文、嵌套线程以及 X Articles（原 Notes）
- PDF 导出支持可选择文本、可点击链接，以及正确渲染 Emoji 和非 ASCII 字符
- 干净的 Markdown，可用于 Obsidian、Logseq、Notion、Hugo 及其他基于 Markdown 的工作流
- 通过 obsidian:// URI 方案一键“添加到 Obsidian”
- 丰富的 YAML frontmatter，包含作者、handle、日期、来源 URL、内容类型和互动统计
- 可选的 Obsidian 友好型 frontmatter：wikilinked [[@handle]] 作者、Dataview 友好元数据，以及生成的标题和描述
- 本地下载嵌入图片，避免链接失效
- 抓取链接卡片的标题、来源域名和预览图
- 抓取投票内容，包括选项、结果百分比以及总票数/状态行
- 保留引用推文的结构和归属信息
- 导出单条推文或整个线程
- x.com 内联导出按钮，以及工具栏弹窗和右键菜单
- 使用 {date}、{handle}、{slug} 和 {type} 等占位符的自定义文件名模板
- 可选的 Obsidian Vault 定向和 Vault 子文件夹支持
- 可选的导出 Markdown 和媒体下载子文件夹
- 多语言界面：简体中文、阿拉伯语、德语、俄语、法语、韩地语、葡萄牙语（巴西）、波斯语、日语、西班牙语、意大利语、英语
- 支持浅色和深色模式

适用于：

- Obsidian 和 PKM 工作流
- 研究与参考资料归档
- AI 提示词和 RAG 流水线
- 构建可搜索的第二大脑
- 离线保存 X 上的长篇内容

当前限制：

- 视频和 GIF 不会导出为可播放的媒体文件
- 如果 x.com 大幅修改页面结构，部分功能可能失效
- 如果在 x.com 标签页已打开时安装或更新扩展，请在导出前重新加载该标签页——这是有意设计的，用于避免未初始化页面上的静默失败
