XClipper هو أداة قص ويب مجانية ومتاحة المصدر لمنصة X (تويتر) — إضافة Chrome تقوم بتنزيل السلاسل والمنشورات والمقالات كـ PDF أو Markdown نظيف لأجل Obsidian والبحث وسير عمل الذكاء الاصطناعي والأرشفة دون اتصال. يعمل بالكامل داخل المتصفح: بدون مفاتيح API، بدون حسابات، بدون تتبع.

تصدير المحتوى بنقرة واحدة:

- حفظ كـ Markdown
- تصدير كـ PDF (نص قابل للتحديد، روابط قابلة للنقر، بدون تحويل إلى صور)
- نسخ Markdown إلى الحافظة
- إرسال مباشرة إلى Obsidian
- تنزيل الصور محلياً بجانب ملف .md

كل شيء يعمل محلياً على جهازك. لا توجد خدمة خلفية، لا تتبع، ولا طلبات خارجية سوى x.com نفسه.

الميزات الرئيسية:

- تصدير التغريدات، السلاسل، التغريدات المقتبسة، السلاسل المتداخلة، ومقالات X (المعروفة سابقاً بـ Notes)
- تصدير PDF مع نص قابل للتحديد وروابط قابلة للنقر ودعم صحيح للرموز التعبيرية والأحرف غير ASCII
- Markdown نظيف يعمل مع Obsidian وLogseq وNotion وHugo وغيرها من سير العمل المعتمدة على Markdown
- دعم بنقرة واحدة لميزة "Add to Obsidian" عبر مخطط URI obsidian://
- frontmatter بصيغة YAML غني يشمل المؤلف، اسم المستخدم، التواريخ، رابط المصدر، نوع المحتوى، وإحصاءات التفاعل
- frontmatter اختياري متوافق مع Obsidian: مؤلفون wikilinked [[@handle]]، وبيانات متوافقة مع Dataview، وعناوين وأوصاف مُولدة
- تنزيل الصور المضمنة محلياً لمنع تلف الروابط
- التقاط بطاقات الروابط مع العنوان، النطاق المصدر، وصورة المعاينة
- التقاط الاستطلاعات مع الخيارات ونسب النتائج وسطر إجمالي الأصوات/الحالة
- الحفاظ على بنية وعزو التغريدات المقتبسة
- تصدير تغريدة واحدة أو سلسلة كاملة
- زر تصدير مدمج داخل x.com، بالإضافة إلى نافذة شريط الأدوات وقائمة النقر بزر الفأرة الأيمن
- قوالب أسماء ملفات مخصصة باستخدام {date} و{handle} و{slug} و{type}
- دعم اختياري لتحديد Vault في Obsidian ومجلدات فرعية
- مجلد فرعي اختياري لتنزيلات Markdown والوسائط
- واجهة متعددة اللغات: العربية، الألمانية، الإنجليزية، الإسبانية، الصينية (المبسطة)، الفرنسية، الهندية، الإيطالية، اليابانية، الفارسية، البرتغالية (البرازيل)، الروسية
- دعم الوضعين الفاتح والداكن

مناسب لـ:

- سير عمل Obsidian وPKM
- البحث وأرشفة المراجع
- مطالبات الذكاء الاصطناعي وخطوط أنابيب RAG
- بناء عقل ثانٍ قابل للبحث
- حفظ محتوى X الطويل دون اتصال

القيود الحالية:

- لا يتم تصدير الفيديوهات وGIF كملفات وسائط قابلة للتشغيل
- قد تتعطل بعض الوظائف إذا غيّر x.com بنية صفحاته بشكل كبير
- إذا تم تثبيت أو تحديث الإضافة مع وجود تبويب x.com مفتوح، يجب إعادة تحميل التبويب قبل التصدير — لتجنب أخطاء صامتة على صفحة غير مهيأة
