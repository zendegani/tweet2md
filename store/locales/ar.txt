XClipper هو إضافة Chrome مفتوحة المصدر تحوّل سلاسل X/Twitter ومنشوراتها ومقالاتها الطويلة إلى Markdown نظيف لأجل Obsidian والبحث وسير عمل الذكاء الاصطناعي والأرشفة دون اتصال.

تصدير المحتوى بنقرة واحدة:

- حفظ كملف Markdown
- نسخ Markdown إلى الحافظة
- إرسال مباشرة إلى Obsidian
- تنزيل الصور محليًا بجانب ملف .md

يعمل بالكامل داخل متصفحك. بدون مفاتيح API، بدون حسابات، بدون تتبع، بدون تحليلات.

الميزات الرئيسية:

- يصدّر التغريدات، السلاسل، التغريدات المقتبسة، السلاسل المتداخلة، ومقالات X الطويلة (المعروفة سابقًا بـ Notes)
- Markdown نظيف يعمل مع Obsidian وLogseq وNotion وHugo وسير العمل الأخرى المبنية على Markdown
- زر "إضافة إلى Obsidian" بنقرة واحدة عبر مخطط URI obsidian://
- frontmatter بصيغة YAML غني يشمل المؤلف، اسم المستخدم، التواريخ، رابط المصدر، نوع المحتوى، وإحصاءات التفاعل
- frontmatter اختياري مهيّأ لـ Obsidian: مؤلفون بصيغة الويكي [[@handle]]، بيانات وصفية متوافقة مع Dataview، وعناوين وأوصاف مُولَّدة
- تنزيل الصور المضمّنة محليًا لتفادي الروابط المعطلة
- التقاط بطاقات الروابط بالعنوان والنطاق المصدر وصورة المعاينة
- التقاط الاستطلاعات بخياراتها ونسب النتائج وسطر إجمالي الأصوات/الحالة
- الحفاظ على بنية وعزو التغريدات المقتبسة
- تصدير تغريدة واحدة أو سلسلة كاملة
- زر تصدير مدمج داخل x.com، إضافة إلى نافذة شريط الأدوات وقائمة النقر بزر الفأرة الأيمن
- قوالب أسماء ملفات قابلة للتخصيص بعناصر نائبة مثل {date} و{handle} و{slug} و{type}
- تحديد اختياري لـ Vault الخاص بـ Obsidian ومجلد فرعي داخل الـ Vault
- مجلد فرعي اختياري لتنزيلات الـ Markdown والوسائط
- واجهة متعددة اللغات: الإنجليزية، الإسبانية، الألمانية، الفرنسية، الإيطالية، الروسية، اليابانية، البرتغالية (البرازيل)، الصينية (المبسطة)، الهندية، العربية، والفارسية
- الوضع الفاتح والداكن

ملائم لـ:

- سير عمل Obsidian وأنظمة PKM
- البحث وأرشفة المراجع
- مطالبات الذكاء الاصطناعي وخطوط أنابيب RAG
- بناء "عقل ثانٍ" قابل للبحث
- حفظ محتوى X الطويل دون اتصال

القيود الحالية:

- لا يتم تصدير الفيديوهات وصور GIF كملفات وسائط قابلة للتشغيل
- قد تتعطل بعض الوظائف إذا غيّر x.com بنية صفحاته بشكل كبير
- إذا قمت بتثبيت أو تحديث الإضافة بينما كان لديك تبويب x.com مفتوحًا، أعد تحميل التبويب قبل التصدير — هذا متعمَّد لتفادي الإخفاقات الصامتة على صفحة غير مهيّأة
