XClipper è un web clipper gratuito con codice sorgente disponibile per X (Twitter) — un'estensione Chrome che scarica thread, post e articoli come PDF o Markdown pulito per Obsidian, ricerca, workflow AI e archiviazione offline. Funziona interamente nel browser: nessuna API, nessun account, nessun tracciamento.

Esporta i contenuti con un clic:

- Salva come Markdown
- Esporta come PDF (testo selezionabile, link cliccabili, senza rasterizzazione)
- Copia Markdown negli appunti
- Invia direttamente a Obsidian
- Scarica le immagini localmente insieme al file .md

Tutto funziona localmente sul tuo dispositivo. Nessun backend, nessuna telemetria, nessuna richiesta esterna oltre x.com.

Funzionalità principali:

- Esporta tweet, thread, tweet citati, thread annidati e X Articles (ex Notes)
- PDF con testo selezionabile, link cliccabili e rendering corretto di emoji e caratteri non ASCII
- Markdown pulito compatibile con Obsidian, Logseq, Notion, Hugo e altri workflow basati su Markdown
- Supporto "Aggiungi a Obsidian" con un clic tramite schema URI obsidian://
- Frontmatter YAML ricco con autore, handle, date, URL sorgente, tipo di contenuto e metriche di engagement
- Frontmatter opzionale ottimizzato per Obsidian: autori con wikilink [[@handle]], metadati compatibili con Dataview e titoli/descrizioni sintetizzati
- Download immagini incorporate in locale per evitare link rotti
- Cattura link card con titolo, dominio sorgente e immagine di anteprima
- Cattura sondaggi con opzioni, percentuali e totale voti/stato
- Mantiene la struttura e l'attribuzione dei tweet citati
- Esporta un singolo tweet o un intero thread
- Pulsante di export inline dentro x.com, popup nella toolbar e menu contestuale
- Template nomi file personalizzabili con {date}, {handle}, {slug}, {type}
- Supporto opzionale vault Obsidian e sottocartelle
- Sottocartella opzionale per download di Markdown e media
- Interfaccia multilingue: Italiano, Arabo, Cinese (semplificato), Francese, Tedesco, Hindi, Inglese, Persiano, Portoghese (Brasile), Russo, Spagnolo, Giapponese
- Modalità chiara e scura

Ideale per:

- Workflow Obsidian e PKM
- Ricerca e archiviazione di riferimenti
- Prompt AI e pipeline RAG
- Costruzione di un secondo cervello ricercabile
- Conservazione offline di contenuti lunghi di X

Limitazioni attuali:

- Video e GIF non vengono esportati come file multimediali riproducibili
- Alcune funzioni possono smettere di funzionare se x.com cambia significativamente la struttura della pagina
- Se installi o aggiorni l'estensione con una scheda x.com già aperta, ricarica la scheda prima dell'esportazione per evitare errori su pagina non inizializzata
