XClipper, X (Twitter) के लिए एक निःशुल्क और स्रोत-उपलब्ध वेब क्लिपर है — एक Chrome extension जो threads, posts और articles को PDF या clean Markdown में डाउनलोड करता है, Obsidian, research, AI workflows और offline archiving के लिए। पूरी तरह आपके browser में चलता है: कोई API keys नहीं, कोई accounts नहीं, कोई tracking नहीं।

एक क्लिक में content export:

- Markdown के रूप में सेव करें
- PDF के रूप में export करें (selectable text, clickable links, बिना rasterization)
- Markdown को clipboard में copy करें
- सीधे Obsidian में भेजें
- images को .md file के साथ locally download करें

सब कुछ आपके device पर locally चलता है। कोई backend service नहीं, कोई telemetry नहीं, x.com के अलावा कोई external request नहीं।

मुख्य फीचर्स:

- tweets, threads, quote tweets, nested threads और X Articles (formerly Notes) export करें
- PDF export selectable text, clickable links और emoji/non-ASCII rendering के साथ
- clean Markdown जो Obsidian, Logseq, Notion, Hugo और अन्य Markdown workflows के साथ काम करता है
- one-click "Add to Obsidian" obsidian:// URI scheme के माध्यम से
- rich YAML frontmatter: author, handle, dates, source URL, content type और engagement stats
- optional Obsidian-friendly frontmatter: wikilinked [[@handle]] authors, Dataview-friendly metadata, और synthesized titles/descriptions
- embedded images को locally download करें ताकि link rot से बचा जा सके
- link cards capture करें: title, source domain और preview image
- polls capture करें: choices, result percentages और total votes/status line
- quote tweet structure और attribution preserve करें
- single tweet या पूरा thread export करें
- x.com के अंदर inline export button, toolbar popup और right-click context menu
- custom filename templates: {date}, {handle}, {slug}, {type}
- optional Obsidian vault targeting और subfolder support
- exported Markdown और media के लिए optional downloads subfolder
- Multi-language UI: Hindi, Arabic, Chinese (Simplified), French, German, Italian, Japanese, Persian, Portuguese (Brazil), Russian, Spanish, English
- light और dark mode support

उपयोग के लिए उपयुक्त:

- Obsidian और PKM workflows
- research और reference archiving
- AI prompts और RAG pipelines
- searchable second brain बनाना
- X के long-form content को offline सुरक्षित रखना

मौजूदा सीमाएँ:

- videos और GIFs playable media files के रूप में export नहीं होते
- कुछ features x.com के structure बदलने पर प्रभावित हो सकते हैं
- अगर extension install/update के समय x.com tab खुला हो, तो export से पहले tab reload करें — यह uninitialized page पर silent failures से बचाने के लिए है
