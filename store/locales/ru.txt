XClipper is a free, open-source web clipper for X (Twitter) — расширение Chrome, которое скачивает треды, посты и статьи в виде PDF или чистого Markdown для Obsidian, исследований, AI workflow и офлайн-архивации. Работает полностью в браузере: без API-ключей, без аккаунтов, без трекинга.

Экспорт контента в один клик:

- Сохранить как Markdown
- Экспортировать как PDF (выделяемый текст, кликабельные ссылки, без растеризации)
- Копировать Markdown в буфер обмена
- Отправить напрямую в Obsidian
- Скачать изображения локально вместе с .md-файлом

Всё работает локально на вашем устройстве. Без backend-сервиса, без телеметрии, без внешних запросов кроме x.com.

Ключевые возможности:

- Экспорт твитов, тредов, цитируемых твитов, вложенных тредов и X Articles (ранее Notes)
- PDF с выделяемым текстом, кликабельными ссылками и корректным отображением emoji и не-ASCII символов
- Чистый Markdown для Obsidian, Logseq, Notion, Hugo и других Markdown-воркфлоу
- Поддержка «Add to Obsidian» в один клик через URI-схему obsidian://
- Rich YAML frontmatter с автором, handle, датами, source URL, типом контента и метриками вовлечённости
- Опциональный Obsidian-friendly frontmatter: авторы в виде wikilink [[@handle]], Dataview-метаданные и синтезированные заголовки и описания
- Локальная загрузка встроенных изображений для предотвращения битых ссылок
- Захват link cards с заголовком, доменом источника и preview image
- Захват опросов с вариантами, процентами результатов и строкой общего числа голосов/статуса
- Сохранение структуры и атрибуции цитируемых твитов
- Экспорт одного твита или всего треда
- Inline-кнопка экспорта прямо внутри x.com, а также toolbar popup и контекстное меню
- Настраиваемые шаблоны имён файлов с {date}, {handle}, {slug}, {type}
- Опциональная поддержка vault Obsidian и подпапок vault
- Опциональная подпапка для загрузок Markdown и медиа
- Многоязычный интерфейс: русский, арабский, китайский (упрощённый), французский, немецкий, хинди, английский, персидский, португальский (Бразилия), итальянский, испанский, японский
- Светлая и тёмная тема

Идеально для:

- Workflow в Obsidian и PKM
- Исследований и архивирования источников
- AI-промптов и RAG-пайплайнов
- Создания поискового «второго мозга»
- Офлайн-сохранения длинного контента X

Текущие ограничения:

- Видео и GIF не экспортируются как воспроизводимые медиафайлы
- Некоторые функции могут не работать при значительных изменениях структуры x.com
- Если расширение установлено/обновлено при уже открытой вкладке x.com, перезагрузите вкладку перед экспортом
